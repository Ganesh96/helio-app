Step 1: Get Your NREL API Key

To use this application, you first need a free API key from the National Solar Radiation Database (NSRDB).

Follow these instructions:

  1. Navigate to the NREL API signup page: https://developer.nrel.gov/signup/
  2. Fill out the required information on their form.
  3. Agree to the terms and conditions and submit the form.
  4. You will receive your API key via email. Copy it to use in the next step.

Solar Panel Arrangement Analysis

Enter location data and your NREL API key to analyze panel performance.

Use the email you used for the NREL API key signup.
